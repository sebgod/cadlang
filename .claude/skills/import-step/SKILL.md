# Import a STEP file into cadlang

## Ad-hoc (outside a project)

```bash
python stepimport.py thing.step                          # just print report
python stepimport.py thing.step -o parts/                # emit .g.cad.py per body
python stepimport.py thing.step --name Foo -o parts/     # custom base name
python stepimport.py thing.step -o parts/x.g.cad.py      # single-body output
```

Each `MANIFOLD_SOLID_BREP` in the STEP becomes one `<Name>_partNNN.g.cad.py`. The importer classifies each body:

- **revolve**: ID/OD cylinder pair + Z-axial → `d.revolve(...)` with extracted radii, height, and radial bore patterns (including `Circular` pattern count when opposite-pair heuristic detects it).
- **extrude**: ≥2 Z-normal planes, not axisymmetric → stacked-layer `d.extrude(...)` calls from Z-level detection, plus axial through-holes and lateral-axis cylinders as `cut(...)`.

Unclassifiable bodies still emit — the output just won't be as clean.

## Inside a project (recommended)

If the project's `project.cadlang` has an `import:` step for this STEP file, run:

```bash
cadlang build
```

from the project directory. The importer runs automatically as part of the pipeline. See `example-project/SWQ8-Dewshield-Holder/project.cadlang` for a reference manifest.

## After import

1. Open each generated `.g.cad.py` and review — the report on stdout flags bodies where classification was ambiguous.
2. If a body is misclassified or needs manual tuning, **don't** edit the `.g.cad.py` in place — those files are regenerated by `cadlang build`. Instead:
   - Copy to a hand-written `<name>.cad.py` (no `.g.` infix), tune there.
   - Update the project to reference the hand-written version.
3. Note measurements / decisions / open questions in the project's `STATE.md`.

## Limitations (from CLAUDE.md)

- Assumes a single global coordinate frame — nested `SHAPE_REPRESENTATION` transforms are not resolved yet. True for typical single-body Fusion exports.
- Extrude bodies currently emit as bbox rectangles per Z-layer (not full profile reconstruction).
- Fillets and chamfers are not extracted as ops.
